## 🚨 COMMON ISSUES & SOLUTIONS

| Issue | Solution |
|-------|----------|
| Port 8000 in use | Use `--port 8001` or `netstat -ano \| findstr :8000` |
| Module not found | Ensure venv is activated and requirements installed |
| Model not found | Run `python src/model_trainer.py` first |
| Docker build fails | Try `docker system prune -a` then rebuild |
| K8s deployment fails | Check `kubectl logs`, ensure image is built |
| MLflow not showing | Run `mlflow ui` in separate terminal |
